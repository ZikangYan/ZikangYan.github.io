 你当前仓库已经配置了 GitHub Pages 自动部署：推送到 main 会触发 Actions 部署。                                                                       
  实现“更新 blog”就是：改内容 → 本地检查 → 提交推送 → 等待 Actions 完成。 

pnpm new-post my-post # 创建新的文章 my-post 不用加.md

本地检查： pnpm check   pnpm build
